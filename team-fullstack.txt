## SECURITY STANDARDS (December 2025)

All agents enforce:
1. ❌ No `dangerouslySetInnerHTML`
2. ✅ CSP headers required
3. ✅ Parameterized queries only
4. ✅ Input validation everywhere
5. ✅ Output encoding for XSS
6. ❌ No hardcoded secrets
7. ✅ Dependency scanning
8. ❌ No deprecated crypto (MD5, SHA1)
9. ✅ Rate limiting on APIs
10. ✅ Proper error handling
11. ✅ Resilience testing (circuit breakers)
12. ✅ Penetration testing (fuzzing, IDOR)
